发布应用

个人中心
  1. 在GitHub创建项目
  2. 项目命名规则:ext-主要语言-应用名称
  3. README.md中有应用详细安装使用教程
  4. 在管理面板个人中心创建应用填写详细信息
  5. 下载链接填写GitHub项目,安装时会获取指定目录下的文件(忽略3层以上目录)
  6. 申请的应用需要公开可见,密钥为空,贡献值为空
  • 项目要有实际应用功能,只有单一功能的,同一语言已有实现项目的不通过
  • 不得骚扰、敷衍、雷同、作弊
  • 不得违反现行法律法规
  • 不得有影响用户体验、危及平台安全及损害他人权益的行为
欢迎开发群管助手类应用,其他应用可能审核不通过
应用发布审核公示之后获取500贡献值
